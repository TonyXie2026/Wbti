# 给 Codex 的开发指令

请根据本项目文档完成一个可运行的前端网页，参考 `sbti.fancc.de5.net` 的体验节奏，但不要复制原站内容。

## 开发目标

做一个“网络梗身份测试”网站，必须包含：

1. Landing 首页
2. 点击开始测试后进入题目
3. 12 道单选题
4. 根据答案计算 16 个标签中的一个结果
5. 结果页展示标签、英文代号、简介、角色图占位
6. 支持重新测试
7. 页面适配手机端和桌面端

## 页面要求

### 首页

- 主标题：测测你是哪种互联网人
- Slogan：互联网自有一套分人方法
- 辅助文案：不一定科学，但很适合认领
- 主按钮：开始测试

### 测试页

- 每次只显示一题
- 每题四个选项
- 点击选项后自动进入下一题
- 顶部显示进度条与题号
- 题目答完后自动跳结果页

### 结果页

- 文案：你的互联网身份是
- 大号标签名
- 英文代号
- 角色图占位
- 一句话简介
- 再测一次按钮

## 数据要求

- 使用 `src/data/tags.json`
- 使用 `src/data/questions.json`
- 不要把内容写死在组件里

## 结果算法

- 每个选项绑定一个主标签
- 统计 12 题得分
- 得分最高者为最终结果
- 若出现并列，优先取最后一次命中的标签，或自行设计 tie-break 规则并写清楚

## UI 要求

- 风格简洁
- 结果卡片可截图
- 不做复杂动画
- 使用圆角白卡 + 浅色背景
- 首页 Slogan 要醒目
- 结果页标签字号最大

## 额外建议

- 角色图组件需要支持图片加载失败时回退到占位图
- 可在 localStorage 保存最近一次结果
- 可增加“复制结果文案”按钮

## 交付标准

- 本地运行无报错
- 首页到结果页完整闭环
- 样式统一
- 数据文件可维护
- 后续方便替换题库与标签图
